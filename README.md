# MathHub (available [here](https://jezzerk.github.io/MathHub/))

A comprehensive web-based math training platform that helps students master mathematics from basic arithmetic to advanced calculus. Perfect for self-study, homework practice, and skill building.

## What It Does

Practice your math abilities in 4 crucial areas:

*Arithmetic Practice
*Unit Circle Practice
*Polynomial Factoring
*Derivatives Practice

## How to Use

### Navigation
- Click any module card to start practicing
- Use "Back to Modules" to return to the main menu
- Click "New Question" for fresh problems
- Adjust challenge level in the controls

### Progress Tracking
- Total number of problems attempted
- Number of correct responses
- Percentage of correct answers
- Statistics update after each question
